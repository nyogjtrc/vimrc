## align

* ``\t=`` Align assignments.
* ``\t,`` Align on commas.
* ``\tsp`` Align on whitespace.
* ``\acom`` Align comments.
* ``\Htd`` Align HTML tables.

## nerdcommmenter

* ``\cc``  Add general comment.
* ``\cm``  Add block comment.
* ``\cu``  Remove comment.
